Personal

個人 Resume 作業目標:

  1. 讓學員透過 GMail 或 Microsoft 帳號, 以個人手機門號登記 Openai 帳號, 可取得一次性 $18 的使用權限, 以 API 程式與 ChatGPT 對談.
  2. 讓學員整理個人的英文版 Resume.
  3. 讓學員了解除了 Word 之外, 也可以使用 LaTeX 程式產生 PDF 文件.
  4. 讓學員了解利用純文字表達文件格式的 LaTeX 檔案, 與 Github 結合後, 應該比 Word 更適合用於協同專案.

個人作業名稱為 resume, 以下列倉儲作為 Template:

https://github.com/mdecycu/Resume-Automation-using-ChatGPT

利用 Github Classroom 建立個人作業, 以 Python、LaTeX 與 ChatGPT 協助建立個人 resume, 最終希望能用來協助建立個人網頁或分組專題報告.

Resume-Automation-using-ChatGPT 倉儲:

  1. 所提供的 Template 只支援英文, 可嘗試利用 LaTeX 程式語法修改為中文
  2. OpenAI 的使用, 可以利用 Gmail 帳號申請 API key, 每一個手機門號只能取得第一個登錄帳號下的 $18 免費使用, 且必須在三個月內用完. 其中使用 Davinci  模型每 1k token 收費 $0.02.
  3. Resume 內容整理使用 Davinci 模型, 可在免費額度內進行測試 (token 與費用計算)
  4. 必須要在可攜程式環境中以 MikTeX (約 800MB) 或 TeXLive (約 2GB) 進行編譯 (納入 Portable MiKTeX 之後的 可攜 Python 程式套件: wcm_portable_w2-4-miktex.7z)
  5. Python 必須安裝 openai module
  6. 利用 tkinter 建立的 GUI 介面表單輸入 Resume 欄位資料後, 交由 OpenAI

個人作業倉儲名稱: mdecd2023/resume-github_帳號, 例如: https://github.com/mdecd2023/resume-scrum-1 就是帳號為 scrum-1 用戶所取得的 resume 作業倉儲.

作業討論:

採 Bing Microsoft Translator 直接將 Why LaTeX? 翻譯如下:

為什麼選擇 LaTeX？

對於包含多個或兩個數學表達式的文檔，LaTeX 為教師提供了比 Microsoft Word 等編輯器更多的優勢。

  1. 在 LaTeX 中包含數學運算式涉及鍵入一些適當的字元。相比之下，在 Word 中包含數學需要公式編輯器，這是一個繁瑣且緩慢的圖形用戶介面。
  2. 由於內容是在純文本檔中指定的，因此通常可以使用搜索和替換或其他節省時間的機制有效地處理在 Word 中費力的任務。
  3. 由於上述原因，LaTeX 可節省大量長期運行時間。
  4. 許多用戶發現，隨著時間的推移，LaTeX 中的排版變得越來越有趣和方便。相比之下，經驗表明，即使對於專家用戶來說，Word的挫敗感在很大程度上仍然存在。
  5. 由於 LaTeX 是開源的，因此您可以輕鬆地與其他教師共用您的內容，而不會出現版本相容性或許可問題。
  6. 由於 LaTeX 已經適應了數學和科學研究社區的需求，因此您不太可能希望在 LaTeX 中無法實現的符號。
  7. 重複性任務可以更容易地自動化。例如，通過適當的設置，可以為學習時程表的學生製作練習表幾秒鐘。
  8. LaTeX 足夠受歡迎，論壇和其他互聯網資源為那些對高級功能感興趣的人提供了良好的支援。

值得一提的是幾個缺點:

  1. LaTeX 有一個學習曲線。這是放棄圖形使用者介面（如 Word 的公式編輯器）所帶來的效率的代價，起初成本肯定很大。
  2. 如果您在代碼中犯了足夠重要的錯誤，LaTeX 將為您提供錯誤訊息而不是輸出檔。但是，調試時間會隨著經驗的增加而迅速減少。

OpenAI Python API 程式範例:

# 導入 openai
import openai

# 輸入 api_key
openai.api_key = "your_key"

# 三個月內有 $18 可以進行測試 Davinci 0.02/1k token


# 利用 create 建立問答
response = openai.Completion.create(
model="text-davinci-003",
prompt="請列出不要讀機械設計系的 5 個理由",
max_tokens=256,
temperature=0,
top_p=1,
frequency_penalty=0,
presence_penalty=0
)

# 傳回的 dict 中的 text
output = response["choices"][0]["text"]
print(output)

"""
1. 機械設計系的課程複雜，學習負擔大。
2. 機械設計系的就業前景不明朗。
3. 機械設計系的課程需要花費大量時間在實驗室中。
4. 機械設計系的課程需要掌握大量的專業知識。
5. 機械設計系的課程需要掌握大量的計算機軟件。
"""

wcm_portable_w2-4-miktex.7z 中的 MiKTeX 安裝 cjk 與 cjk-fonts 後, 可以利用:

pdflatex -interaction=nonstopmode chinese_ex1.tex 將 chinese_ex1.tex 轉為 pdf 檔案.

chinese_ex1.tex 內容如下:

% 轉檔指令: pdflatex -interaction=nonstopmode chinese_ex1.tex
\documentclass[book, oneside, 12pt]{article}
\usepackage{CJKutf8}
\begin{document}
%以下 bsmi 為明體, 若要使用楷書, 則使用 bkai
\begin{CJK*}{UTF8}{bsmi}

\section{標題}
網際內容管理與協同產品設計實習 \\
Web-based content management, collaborative product design.

\section{2023 Spring 學期}
安裝 cjk 與 cjk-fonts 終於可以使用中文

\end{CJK*}
\end{document}

修改上述 Resume.tex 只要加入 \usepackage{CJKutf8}, 並在 \begin{document} 之後加上 \end{document} 成為:

\begin{document}
\begin{CJK*}{UTF8}{bsmi}

以及在 \end{document} 之前加入 \end{CJK*} 成為:

\end{CJK*}
\end{document}

Resume.tex 內文就可以加入中文, 並以 pdflatex -interaction=nonstopmode Resume.tex 轉為 Resume.pdf
